Instructions

Input section

If you want a man of the match you have to enter atleast one player

The Team names are not required to be entered, if you don't enter them it will show up with home and away

Scoreboard Section

Change Home Team Score

Score up in the set - W key

Score goes down in the set - S key

Sets goes up - T key

Sets goes down - G key

Change Away Team Score

Score up in the set - I key

Score goes down in the set - K key

Sets goes up - Y key

Sets goes down - H key
